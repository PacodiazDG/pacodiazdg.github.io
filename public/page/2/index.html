Squid proxy 6.0 for Https Filter [Ubuntu 24.04 LTS] [Transparent Proxy]

2025-06-07
#Squid-Proxy  #SNI  #TLS  #Web-Filter  #SSL-Inspection 
Squid Proxy is a great tool that allows us to do multiple things; one of them is to function as a forward proxy, and with the use of ACLs, we can filter sites using the SNI. installation Squid proxy is installed via apt-get. The squid-openssl version is of interest to us since it contains ssl-bump. If our distribution does not include squid-openssl, it must be installed manually. sudo apt-get install squid-openssl We generate the SSL certificates, remember they must be in /etc/squid/ssl; otherwise, Squid will not trust the certificate.
